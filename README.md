# Peliy11 Auto Ally

simple fast roblox auto ally bot. sends ally requests using your cookies without proxies or webhooks.

## setup

1. put your roblox cookies in `config.json` inside the `"cookies"` array
2. put your target group id in `config.json` under `"group_id"`
3. run `python main.py`

## requirements
- python 3
- `pip install aiohttp`
